# Summarization Skill

When summarizing content:

1. **Lead with the key takeaway** (1-2 sentences)
2. **Main points** (3-5 bullet points)
3. **Details** (only if requested or critical)
4. **Source attribution** (when applicable)

Keep summaries under 200 words unless asked for more detail.
