# Creative Coordinate System

Every ad concept is mapped to coordinates: **Format × Angle × Style × Hook × Funnel**. This system ensures creative diversity, grounds ideation in structure, and tracks what's been tested vs what's unexplored.

## Styles (apply to any Format × Angle)

| Style | Description |
|-------|-------------|
| Lo-fi / Organic | Phone footage, natural light, imperfect — feels like organic content |
| Hi-fi / Polished | Studio quality, professional editing, branded |
| Energetic / Fast | Quick cuts, high energy, trending music |
| Calm / Soft | Slow pace, ASMR-adjacent, ambient |
| Funny / Comedic | Humor-driven, entertainment-first |
| Emotional / Story | Narrative arc, feelings-driven |
| Clinical / Scientific | Data-driven, clean, factual |
| Aspirational / Luxury | Premium feel, lifestyle-first |

## Hook Types (first 3 seconds)

Pattern Interrupt, Curiosity Gap, Social Proof, Problem Agitation, Story, Authority/Credibility. See `meta-ads-strategy` skill for detailed hook frameworks.

## Notation

```
F01:Talking Head × A01:Problem→Solution × Lo-fi × Curiosity Gap × TOF
```

## Proven Combinations

- **F01 × A01** (Talking Head + Problem→Solution): Workhorse for TOF. Lo-fi outperforms hi-fi.
- **F01 × A14** (Talking Head + Testimonial): Strong for MOF. Authenticity dial high.
- **F03 × A09** (Product Demo + Science): Works for ingredient/tech brands.
- **F05 × A07** (VO + B-roll + Lifestyle): Premium brand awareness. Hi-fi style.
- **F16 × A01** (No-Ads Ad + Problem→Solution): Highest CTR for organic-feeling TOF.
- **F14 × A02** (Static + Social Proof): Reliable for retargeting/BOF.
- **F15 × A06** (Carousel + Education): Strong engagement, education-first brands.

## Untested / Emerging

- **F02 × A04** (Interview + Founder Story): High potential for brand trust.
- **F06 × A05** (Split Screen + Comparison): Underused — strong differentiation play.
- **F17 × A01** (Two-Person + Problem→Solution): Trending TikTok format.
- **F10 × A09** (ASMR + Science): Niche but compelling for sensory products.

## Diversity Rules

- When generating 5+ concepts, use at least 3 different formats.
- No single format should exceed 60% of a client's active ad spend.
- Track gap matrix: format × angle combos never tested for this client.
- Prioritize untested combos that have proven track records for similar clients.
